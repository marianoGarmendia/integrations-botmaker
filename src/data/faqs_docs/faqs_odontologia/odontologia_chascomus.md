# Odontología - Chascomús

## Convenio Odontológico

### Círculo Odontológico de Chascomús
Tenemos convenio con el **Círculo Odontológico de Chascomús**. Podés atenderte con los profesionales que pertenecen a dicha institución.

## Preguntas Frecuentes

### ¿Con qué odontólogos me puedo atender?
**Respuesta:**
Tenemos convenio con el **Círculo Odontológico de Chascomús**, por lo tanto podés atenderte con los profesionales que pertenecen a dicha institución.

### ¿Qué odontólogos tengo con mi plan?
**Respuesta:**
Todos los odontólogos que pertenecen al Círculo Odontológico de Chascomús.

### ¿Dónde consulto la lista de odontólogos?
**Respuesta:**
Podés consultar en la Sucursal Chascomús:
- **Dirección:** Soler 229
- **Teléfono:** 02241-436891

### ¿Necesito autorización previa?
**Respuesta:**
Depende del tratamiento. Para consultas simples generalmente no. Para tratamientos complejos, consultá en la sucursal.
