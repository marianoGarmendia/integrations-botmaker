# Consultas Médicas - Chascomús

## Convenio Médico

### Círculo Médico de Chascomús (CIRMEDC)
PRIMEDIC Salud tiene convenio con el **Círculo Médico de Chascomús (CIRMEDC)**. Podés atenderte con los médicos que pertenecen a esta institución.

## Preguntas Frecuentes

### ¿Con qué médicos puedo atenderme para hacer una consulta?
**Respuesta:**
PRIMEDIC Salud tiene convenio con el **Círculo Médico de Chascomús (CIRMEDC)**. Podés atenderte con los médicos que pertenecen a esta institución.

### ¿Qué es el CIRMEDC?
**Respuesta:**
CIRMEDC es el Círculo Médico de Chascomús, la asociación de médicos con la que PRIMEDIC tiene convenio en la ciudad de Chascomús.

### ¿Puedo elegir cualquier médico del CIRMEDC?
**Respuesta:**
Sí, podés elegir libremente cualquier médico que pertenezca al Círculo Médico de Chascomús.

### ¿Necesito autorización previa para consultas?
**Respuesta:**
En general, las consultas médicas no requieren autorización previa. Para estudios o tratamientos especiales, consultá en la sucursal Chascomús.

### ¿Dónde consulto más información?
